Model Context Protocol - Python Implementation

Quick Start

Get started with the MCP Python SDK in minutes:

# Install MCP uv add "mcp[cli]" # Create a simple server from mcp.server.fastmcp import FastMCP mcp = FastMCP("Demo") @mcp.tool() def add(a: int, b: int) -> int: """Add two numbers""" return a + b # Run the server mcp.run()

Core Features

  • Tools: Expose functions to LLMs for execution
  • Resources: Provide data and context to LLMs
  • Prompts: Define reusable prompt templates
  • Multiple Transports: stdio, SSE, Streamable HTTP

Example: Creating a Tool

from mcp.server.fastmcp import FastMCP mcp = FastMCP("Calculator") @mcp.tool() def calculate(operation: str, x: float, y: float) -> float: """Perform basic arithmetic operations""" if operation == "add": return x + y elif operation == "subtract": return x - y elif operation == "multiply": return x * y elif operation == "divide": return x / y if y != 0 else float('inf') else: raise ValueError(f"Unknown operation: {operation}")

Multi-Language Integration

The MCP Python SDK can be integrated with various languages and platforms:

  • Python (native)
  • JavaScript/TypeScript (via stdio transport)
  • Ruby (via command execution)
  • Kotlin/Java (via ProcessBuilder)
  • PowerShell (via Python invocation)
